# Quiz App

A clean, modern, and responsive multiple-choice quiz app built as a beginner project for my developer portfolio. It shows one question at a time, gives instant color feedback on answers, counts your score, and displays a final result screen with a restart option. Built with only HTML, CSS, and vanilla JavaScript.

## Features

- 5 multiple-choice questions stored in a JavaScript array
- One question shown at a time
- Instant feedback (correct answer in green, wrong pick in red)
- Score counting
- Final result screen with a **Restart** button
- Responsive design with hover effects on options

## What I Learned

- How to store data in a **JavaScript array of objects**
- How to **create elements** (buttons) with JavaScript
- How to handle **click events** and check answers
- How to **track state** (current question, score)
- How to **show and hide screens** by toggling CSS classes
- How to make a UI **responsive** with hover effects
